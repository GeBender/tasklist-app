# Tasklist APP
Simple Tastklist app - Silex / Doctrine / Bootstrap

### Instalação ###
* Clone este repo
* Instale as dependências via Composer

`composer install`


### Testando ###
Levante a aplicação em um webserver, ex:

`php -S 0.0.0.0:8080`


Execute a aplicação em um navegador
http://localhost:8080
